Initialize base script for recaptcha

		$loader->load('recaptcha')->init();
Make sure you specify a site key on the recaptcha config, otherwise, you can always provide it as a first argument when you call the load recaptcha box

Load Recaptcha Box

		$loader->load('recaptcha')->loadRecaptchaBox("[site key]","[success callback]","[expired callback]","[error callback]");
[success callback] - called when user successfully completed recaptcha frontend validation [expired callback] - called when recaptcha expires [expired error] - called when recaptcha has an error

To verify recaptcha from your backend, you can call verify_recaptcha function

		$loader->load('recaptcha')->recaptcha_verify([response],[secret key])
[response] - hidden input with name "recaptcha_response" were automatically added unto your form that house the recaptcha so you can get the response value and pass as a response argument [secret key] - this comes with your site key (google recaptcha admin console)
